# refactoring_2nd

## 📋 요구 사항
### 개발 환경
- Java 23
- Gradle 빌드 도구

---

## ▶️ 실행 방법
1. **프로젝트 복제**:
   ```bash
   git clone https://github.com/silverhyuk/refactoring_2.git
   cd refactoring_2
    ```
2.	Gradle 빌드:
   ```bash
   ./gradlew build
   ```
3.	프로그램 실행:
   ```bash
   ./gradlew run
   ```
